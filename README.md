## 📌 Featured Project

A clean, TypeScript-first rebuild of a call-center / dialer admin dashboard,
built on the **Next.js App Router**.

- ⚡ **Next.js 16 (App Router)** + **TypeScript (strict)** with a secure
  server-side proxy layer — the browser only ever talks to Next route handlers
  (access tokens live in httpOnly cookies, never in the browser)
- 🔄 **TanStack Query** for data fetching, caching, and request cancellation
- 🎨 **Tailwind CSS** design system with a single themed palette
- 🧱 Feature-first architecture: Agents, Leads, Contacts, Teams, Channels,
  Call Flows, QA, SMS (Brands / Campaigns / Numbers), Phone Numbers & more
- 🧭 Dedicated add/edit pages, URL-driven table filters, and Zod-validated
  API boundaries

> _Tech: Next.js · TypeScript · TanStack Query · Tailwind CSS · Zod_

🧰 _Beyond this, I've built a range of other projects — landing pages, dashboards, booking platforms, social media and real estate apps, and more._
